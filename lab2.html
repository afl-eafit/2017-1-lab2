Introduction

Kleene’s algorithm transforms a given deterministic finite automaton (DFA) into a regular expression. Together with Thompson’s construction, it establishes the equivalence of several description formats for regular languages [Gross and Yellen, 2004].

Assignment (80%)

  • Download the GitHub repository 2017-1-lab2.
  • Using the data types in the above repository defines a function 
              kleen :: DFA -> RegExp
    that takes a deterministic finite automaton intro a regular expression by implementing Kleen's algorithm [Hopcroft and Ullman, 1979].
